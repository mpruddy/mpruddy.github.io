Indexed Pedegree File

There are various way one can enter this database. I have supplied a "HOME" hyperlink to return to this page. If you find yourself lost in amongst the pedegrees in the database, click the word INDEX which is at the bottom of each page in order to get to the "Index of Persons" page and then at the top of the index page click on HOME PAGE  to get back to this page. 

Above on this page is a link to the "Civil War Home Page" which takes you back to the entry page for my Rootsweb website. 

If you click on MICHAEL GERARD RUDDY you can start a search backwards up the pedegree file into the database. 

If you click on SURNAMES you will be taken into the "Index of Persons" at the surname you click on.

If you click on PERSONS you will be taken to the origin page of the index matrix which contains over 80,000 names broken into five alphabetical from-to segments. Determine alphabetically in which segment the name you are searching lies and click on the appropriate segment to go one lever further into the names, In this manner you can drill down to a specific individual.

NOTE: The term "Living Ruddy" or "Living McNulty" etc. indicates someone whose name has been suppressed because the Gedcom program has determined that person might still be alive.
